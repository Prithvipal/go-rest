### Create a new TODO

**API:** localhost:8080/api/v1/todo<br/>
**Method:** POST<br/>
**Payload**:
```
{
    "value": "Todo Number one"
}
```

**Response:**
```
{
    "msg": "TODO created successfully"
}
```
